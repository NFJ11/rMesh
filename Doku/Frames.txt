

Byte 0: Frame-Typ
Byte 1: Header 0
Byte 2..x: Payload 0

Frame-Typen:
0x00 -> Announce Frane
0x01 -> Annouce Replay Frame


Header:
Bit 7..4 -> Payload Typ:
    0x00 -> Absender Rufzeichen
    0x00 -> Empfänger Rufzeichen

Bit 3...0 -> Typ Rufzeichen -> Länge
Bit 3...0 -> Typ Nachricht -> Nachrichten Typ


Nachrichtentypen:
    0x0 -> Textnachricht an alle
